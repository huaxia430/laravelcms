LaravelCMS
---

这是一个使用Laravel 5.4框架编写的CMS系统，该系统的主要功能有管理员管理、角色管理、文章管理、栏目管理、配置管理、日志管理。


数据库配置
数据库配置在根目录下，有一个laravel_cms.sql文件，默认的管理员账号：admin、密码:123456
